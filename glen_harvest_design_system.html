Design Rules
Border Radius: All interactive elements (buttons, inputs, tags, chips, badges) use border-radius: 9999px (full pill). Cards use 24px. Images use 16px. Never use sharp corners anywhere.
Primary Palette: Forest Green (#2C5440) is dominant. Cream (#F2E8D4) is the secondary surface and text-on-dark. Peri Red (#B83A28) used only for the spicy variant accent. Gold (#C8924A) for premium moments.
Typography: Playfair Display (serif) for all headlines and the brand logo. DM Sans for all body copy, labels, and UI text. Never mix more than these two families.
Pattern: The signature makhana dot texture (radial-gradient dots) is applied as a subtle overlay on green backgrounds. Opacity 6–10% only — atmospheric, never distracting.
Tone: Refined organic luxury. Never clinical or overly modern. Warm, grounded, trustworthy — like a premium farmers market, not a tech startup.
Flavor Colors: Mint Masti = Forest Green system. Peri Peri = Brick Red system. Each flavor gets its own themed section but the base chrome (nav, footer) stays in Forest Green always.
